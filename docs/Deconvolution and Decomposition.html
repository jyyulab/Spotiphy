Decompose ST.

Parameters:

  • adata_st ‚Äì Original spatial transcriptomics data.

  • adata_sc ‚Äì Original single-cell data.

  • key_type ‚Äì The key that is used to extract cell type information from adata_sc.obs.

  • cell_proportion ‚Äì Proportion of each cell type obtained by the deconvolution.

  • save ‚Äì If True, save the generated adata_st as a file.

  • out_dir ‚Äì Output directory.

  • threshold ‚Äì If n_cell is none, discard cell types with proportion less than threshold.

  • n_cell ‚Äì Number of cells in each spot.

  • spot_location ‚Äì Coordinates of the spots.

  • filtering_gene ‚Äì Whether filter the genes in sc_reference.initialization.

  • filename ‚Äì Name of the saved file.

  • verbose ‚Äì Whether print the time spend.

  • use_original_proportion ‚Äì If the original proportion is used to estimate the iscRNA. Note that even when the original proportion is used, we still filter some cells in iscRNA.

Returns:

Anndata similar to scRNA, but obtained by decomposing ST.

Simulation of the spatial transcriptomics data based on a real spatial sample and deconvolution results of Spotiphy.

Parameters:

  • adata_st ‚Äì Original spatial transcriptomics data.

  • adata_sc ‚Äì Original single-cell data.

  • key_type ‚Äì The key that is used to extract cell type information from adata_sc.obs.

  • cell_proportion ‚Äì Proportion of each cell type obtained by the deconvolution.

  • n_cell ‚Äì Number of cells in each spot, either a key of adata_st.obs or a positive integer.

  • batch_effect_sigma ‚Äì Sigma of the log-normal distribution when generate batch effect.

  • zero_proportion ‚Äì Proportion of gene expression set to 0. Note that since some gene expression in the original X is already 0, the final proportion of 0 gene read is larger than zero_proportion.

  • additive_noise_sigma ‚Äì Sigma of the log-normal distribution when generate additive noise.

  • save ‚Äì If True, save the generated adata_st as a file.

  • out_dir ‚Äì Output directory.

  • filename ‚Äì Name of the saved file.

  • verbose ‚Äì Whether print the time spend.

Returns:

Simulated ST Anndata.
